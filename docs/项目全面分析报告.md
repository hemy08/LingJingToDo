## 一、项目整体结构和技术栈

- Rust Edition 2021
- Tauri 2.x（桌面应用框架）
- serde + serde_json（序列化）
- chrono 0.4.44（时间处理）
- rust_xlsxwriter + calamine（Excel处理）

**Tauri 插件**：
- tauri-plugin-opener
- tauri-plugin-dialog
- tauri-plugin-process
- tauri-plugin-fs

### 1.3 架构特点

**前后端分离架构**：
- 前端负责UI渲染和用户交互
- 后端负责数据处理和文件系统操作
- 通过 Tauri IPC 桥梁通信

**数据流架构**：
```
用户操作 → Vue组件 → Pinia Store → API连接层 → Tauri IPC → Rust后端 → 文件系统
```

const pendingRequests = new Map<string, Promise<any>>()
function dedupedRequest(key: string, fn: () => Promise<any>) {
  if (pendingRequests.has(key)) {
    return pendingRequests.get(key)!
  }
  const promise = fn().finally(() => pendingRequests.delete(key))
  pendingRequests.set(key, promise)
  return promise
}
